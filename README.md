# Talk-A-Tive 💬

A real-time full-stack chat application built with the MERN stack.

## Features
- JWT-based authentication (login & signup with profile picture)
- One-on-one and group messaging
- Real-time communication with typing indicators (Socket.io)
- Search users, create/manage group chats
- Message notifications
- Profile viewing

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Real-time:** Socket.io
